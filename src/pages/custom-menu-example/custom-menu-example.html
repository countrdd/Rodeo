Menus

There are 2 type of menus in the starter template

  1. The "Main Menu"
    • Left side menu in the split pane, which by default the mobile menu
    • Menu items are defined in "./app/app.pwa.config.ts" -> MenuItems
    • This menu is declared in the DOM in the "./app/app.html" file
  2. The "Top Navbar Menu"
    • Top navbar menu show on screen sizes larger then 992px
    • This menu is declared in the DOM using the pwa-menu component
    • The menu items are automatically added to the mobile menu when screen size is smaller then 922px

Things to know

The template allows for two main layout styles.

  1. full screen with a split pane
  2. generic centered layout

This has implications on the menu setup. If you set the "generic centered layout" there wont be a main side bar menu in the split pane, however when in a mobile view the main sidebar menu is the main mobile menu. Therefor even if you set a "generic centered layout" the menu items set in the "./app.pwa.config.ts" will still be used as the main moblie menu items. You can see an example of this in action by navigating to the "center" link in this project.
