Center Example
Centered content
Generic center content based style

This layout works best without the split pane and is removed by default. NOTE pwa-center directive will ONLY work by adding a direct ‹ion-grid› child element. 

      ‹ion-content pwa-center›
        ‹ion-grid›

Because you can choose which pages are center align, all other pages will still have the split pane ( this takes into account "web user" view pages and "administration / account holder" view pages) .

Automatic Menu Move
Main Menu To Top Navbar

Notice that the main menu items will automatically be added to the top navbar ( You will NOT have to set these menu items in the MenuItemsTopNav ) items, because all center aligned content removes the split pane the main menu items need to be moved into the visible top navbar.
