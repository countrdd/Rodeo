Max Split Pane Width
Set A Max Width On The Split Pane

Add the pwa-split-pane-max-width directive on the ‹ion-split-pane› tag in "./app/app.html", which allows you to set the maximum width for your split pane side menu. The directive targets the ‹ion-menu› child element on the ion-split-pane tag. 

    ‹ion-split-pane [pwa-split-pane-max-width]="200"›
      ‹ion-menu [content]="content"›

Dont forget to add the menu toggle to your page navbars in order to get the menu toggle on smaller screens 

    ‹ion-header›
      ‹ion-navbar›
    
        ‹button ion-button menuToggle›
          ‹ion-icon name="menu"›‹/ion-icon›
        ‹/button›
Remove Split Pane
On Per Page Basis

You can remove the split pane on certian pages by adding the pwa-center directive to the your pages ‹ion-content› . see the "center" link for more details.
NOTE pwa-center directive will ONLY work by adding a direct ‹ion-grid› child element 

    ‹ion-content pwa-center›
      ‹ion-grid›
Remove Split Pane Completely
Remove The Split Pane From The Application Completely

If you are not using the split pane at all and you jusut want a standard template then you can use the "./app-centered.html" file to replace the current "app.html" ( just rename ). You probably will want to remove the pwa-split-pane-max-width directive as well because there will be no use for it.
