Flat Style
Directive Removes Md ( Android ) Mode Style On Certian Elements

By default the flat style is applied to styles larger then 922px by adding the directive pwa-lg-flat-style on the ‹ion-nav› in "./app/app.html". The idea is to maintain mobile styles from tablet and smaller. Note that if you change the global app mode settings to ios then none of these styles will be changes. The styles are specific to md (android) you can viiew the styles at "./theme/pwa-lg-flat-style.scss"
